# 管理岗位未签劳动合同双倍工资案例集

## 数据来源

元典法律数据库（非公众号数据源），通过语义检索接口采集。

## 采集主题

劳动纠纷中**店长、经理、总经理、主管**等管理类职位劳动者主张未签订书面劳动合同双倍工资差额的裁判案例。

## 案例概况

| 维度 | 数据 |
|------|------|
| 案例数量 | **15篇** |
| 案由 | 劳动争议、劳动合同纠纷 |
| 管理职位类型 | 店长（10篇）、总经理（1篇）、经理/店长（3篇）、行政经理（1篇） |
| 审理层级 | 基层法院（7篇）→ 中级法院（8篇） |
| 地域覆盖 | 12个省级行政区（天津、宁夏、重庆、贵州、湖北、辽宁、江苏、上海、吉林、广东、湖南） |
| 裁判年份 | 2018–2026年 |

## 核心裁判规则分类

### 支持双倍工资的情形（7篇）

| 案例 | 裁判规则 |
|------|----------|
| 宁夏·总经理 | 劳动者担任高管不影响签约义务归属，公司未证明劳动者故意/重大过失 → 支持 |
| 重庆·经理 | 公司未能证明店长职责含合同管理且曾要求签约被拒 → 支持 |
| 贵州·店长 | 小店店长不具独立人事权，签约是用人单位法定义务非劳动者义务 → 支持 |
| 武汉·店长（肖某） | 公司有独立行政人事部门负责签约，店长不具签约自主决策权 → 支持 |
| 武汉·店长（吴某梓） | 协助招聘≠负有人事管理职责，公司管理不规范不能免责 → 支持 |
| 抚顺·店长 | 双方均不能证明过错方，依签约义务法定归属于用人单位 → 支持 |
| 上海·店长 | 内部手册明确店长合同由他人负责，经营困难期间工作性质变更 → 支持 |

### 驳回双倍工资的情形（8篇）

| 案例 | 裁判规则 |
|------|----------|
| 天津·店长 | 明知未签合同法律后果+曾督促他人签合同但自身未签 → 驳回 |
| 武汉·店长（李某） | 实际参与人事管理+知晓签合同要求+未证明曾要求签被拒 → 驳回 |
| 沈阳·店长 | 收到合同文本后仅提一次要求即不再跟进 → 认定自身过错 → 驳回 |
| 常州·店长 | 负有签约管理职责却未尽义务+未证明被拒 → 违反诚信原则 → 驳回 |
| 重庆·店长（唐某） | 自认有人事管理职责+未证明曾主动要求签合同 → 驳回 |
| 长春·店长 | 公司自认店长不负责人事/不保管公章 → 支持 |
| 惠州·行政经理 | 公司对劳动者职位描述前后不一致 → 举证不足 → 支持 |
| 岳阳·店长 | 用人单位应签合同是法定义务，店长身份不免除 → 支持 |
